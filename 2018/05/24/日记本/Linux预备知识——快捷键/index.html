Linux预备知识——快捷键

发表于 2018-05-24

登录

ssh root@ip地址
// 然后输入密码

网络管理命令

systemctl status httpd
systemctl status apache2
systemctl start apache2 // 启动
systemctl stop apache2 // 停止
enable // 跟随系统启动一起启动
disable // 不跟随系统一起启动
ifconfig // 查看ip地址
ip addr // 查看ip地址
route // 路由信息，网关

命令行的下载命令：

wget 下载地址
curl 下载地址 -o 写入到指定名称的文件 --progress
// 现实进度条 --progress

查看服务器时间

date

查看Linux 帮助

curl --help
man curl //按q退出

在终端不小心按了 ctrl + s 怎么办？？

ctrl + s // 暂停屏幕输出
使用
ctrl + q // 恢复屏幕输出

其它快捷键

ctrl + d // 结束输入或退出shell
ctrl + l //（小写的L，不是I）清屏，等同于clear
ctrl + a / ctrl + e // 快速移动光标到行首/行尾

进入vim后，退出时 提示没有文件名
:q! // 不保存，强行退出
